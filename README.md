# Personal Blog

A complete personal blog website built with Next.js, TypeScript, and Tailwind CSS.

## Included

- Public homepage and journal archive
- Single post pages
- Category and tag routes
- About and contact pages
- Admin dashboard with posts, media, comments, and settings screens
- Sitemap, robots, and RSS feed

## Run

```bash
npm install
npm run dev
```
